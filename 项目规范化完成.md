## 🚀 快速上传到 GitHub

### 方式 1: 使用自动化脚本（推荐）

```bash
cd cloudflare-analytics

# 运行准备脚本
./scripts/prepare-github.sh
```

这个脚本会自动：
- ✅ 检查开发环境
- ✅ 运行代码检查
- ✅ 检查敏感文件
- ✅ 验证必要文件
- ✅ 整理文档（可选）
- ✅ 提交代码（可选）

### 方式 2: 手动上传

#### 步骤 1: 初始化 Git（如果还没有）
```bash
cd cloudflare-analytics
git init
```

#### 步骤 2: 添加所有文件
```bash
git add .
```

#### 步骤 3: 首次提交
```bash
git commit -m "feat: initial commit - Cloudflare Analytics Mobile App"
```

#### 步骤 4: 创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名称: `cloudflare-analytics`
3. 描述: `A powerful mobile app for Cloudflare traffic analytics`
4. 选择 Public 或 Private
5. **不要**勾选 "Initialize this repository with a README"
6. 点击 "Create repository"

#### 步骤 5: 关联并推送
```bash
# 添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/cloudflare-analytics.git

# 推送代码
git branch -M main
git push -u origin main
```
